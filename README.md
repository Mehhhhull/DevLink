# DevLink 🚀

### AI-Powered Hackathon Co-Founder Platform

DevLink is an AI-powered platform designed to help hackathon participants build stronger teams, discover better ideas, receive expert guidance, and improve their chances of winning.

Instead of relying on friendship-based team formation and random idea selection, DevLink leverages AI and community-driven collaboration to help participants make smarter decisions throughout the hackathon journey.

---

## Problem Statement

One of the biggest challenges in hackathons is that participants often:

* Form teams based on friendships instead of complementary skills.
* Struggle to find high-quality project ideas.
* Lack access to experienced mentors.
* Build projects that are over-scoped or technically unrealistic.
* Receive little to no feedback before presenting to judges.

As a result, many teams fail not because of poor talent, but because of poor planning and execution.

---

## Solution

DevLink provides an all-in-one ecosystem for hackathon participants by combining:

* Team Discovery
* Idea Hub
* AI-Powered Hackathon Judge
* Mentor Community

into a single platform.

## 🧠 AI Hackathon Judge

DevLink includes a Multi-Agent AI Evaluation System powered by Gemini.

The system simulates real hackathon judging by evaluating projects across multiple dimensions:

### Innovation Agent

Evaluates:

* Originality
* Differentiation
* Creativity

### Feasibility Agent

Evaluates:

* Technical practicality
* Build realism
* Hidden challenges

### Technical Agent

Evaluates:

* Engineering complexity
* System design
* Technical depth

### Appeal Agent

Evaluates:

* Demo potential
* User excitement
* Judge appeal

### Scope Agent

Evaluates:

* MVP feasibility
* Feature balance
* Hackathon suitability

### Head Judge Agent

Combines all evaluations into:

* Overall Score
* Winning Probability
* Biggest Strengths
* Biggest Weaknesses
* Improvement Suggestions

# AI Judge Workflow

1. User submits project idea.
2. Five AI Judge Agents evaluate the idea in parallel.
3. Each agent generates scores and feedback.
4. Scores are aggregated.
5. Head Judge Agent creates final verdict.
6. Results are displayed to the user.
